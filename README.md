# Smart Finance Tracker 

A web application to track income, expenses, and financial insights.

## Features
- Add and manage transactions
- View income and expense summary
- Category-wise spending insights
- Detect suspicious transactions

## Tech Stack
- React
- TypeScript
- Tailwind CSS

## Description
This project helps users manage personal finances efficiently by tracking transactions and providing insights into spending patterns. It also includes a feature to identify suspicious transactions.

## Future Improvements
- Add authentication
- Store data using database
- Improve AI-based anomaly detection
